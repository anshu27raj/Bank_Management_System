# Bank Management System using C++ and DSA

A console-based **Bank Management System** developed in **C++** to demonstrate the practical implementation of **Data Structures and Algorithms (DSA)** concepts such as **Hash Tables, Linked Lists, Queues, and Priority Queues**.

## Features

- Create new bank accounts
- Deposit money into accounts
- Withdraw money from accounts
- View account details
- Maintain transaction history
- Add pending transactions
- Process pending transactions
- Display top account holders based on balance

## Technologies Used

- C++
- STL (Standard Template Library)
- Object-Oriented Programming
- Data Structures and Algorithms
- Visual Studio Code
